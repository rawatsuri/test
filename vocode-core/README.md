# Vocode Core

This directory contains the Vocode voice AI library for building voice-based LLM applications.

Vocode is an open source library that makes it easy to build voice-based LLM apps. You can build real-time streaming conversations with LLMs and deploy them to phone calls, Zoom meetings, and more.

## Installation

```bash
pip install vocode
```

For full documentation, visit: [docs.vocode.dev](https://docs.vocode.dev/open-source)
